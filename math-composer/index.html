Special operators and Symbols

In Math Composer, everything you type can be categorized into two things: "symbols" and "special operators".

  Symbol Special operator
What it is Single element Group of one or more elements, which can be "symbols". You can also have nested special operators inside special operators.
What it looks like Black text Blue text
Examples \(1, x, +, >, \Sigma\) \(\sqrt{x}, x^2, \frac{x}{y}\)

How to select "symbols" or "special operators":

- Use the left and right buttons to move through the equation.

- The current selected element is highlighted in blue.

How to edit "special operators":

- Select the special operator as shown above

- Press down arrow to select an input

How to navigate within "special operators":

- Use the left and right buttons to move through the "inputs" of the "special operator".

How to add elements:

- Pressing any of the buttons on the keyboard will add that "symbol" or "special operator" behind the currently selected element.

How to delete elements:

- If you have made a typo or want to edit one of the "inputs", you can use the backspace button to delete the "symbol" or "special operator" that is highlighted in blue.
