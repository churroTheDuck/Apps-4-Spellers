Tutorial

Layout

The equation display area is where you can view the equations that you type.

The edit menu allows you to share your equation, change your keyboard, delete elements, or add new lines to the equation display area.

The modular keyboard changes depending on the current keyboard selected using the edit menu.

Units and Symbols

In Math Composer, everything you type can be categorized into two things: "units", and "symbols".

  Symbol Unit
What it is Single element Group of one or more elements, which can be "symbols". You can also have nested units inside units.
What it looks like Black text Blue text
Examples \(\sqrt{x}, x^2, \frac{x}{y}\) \(1, x, +, >, \Sigma\)

How to select "units" or "symbols":

Use the left and right buttons to move through the equation. The current selected element is highlighted in blue.

How to edit "units":

If you just created a new "unit", such as a fraction, you might want to change its values. Before editing the "inputs" of a "unit", you must first select it as shown above. Once it is selected, press the down button to edit its "inputs".

How to navigate within "units":

Now that you have selected the "unit" you want to edit, you need to navigate through the "inputs" to find the correct one you want to change. Use the left and right buttons to move through the "inputs" of the "unit".

How to add and delete elements:

Pressing any of the buttons on the keyboard will add that "symbol" or "unit" behind the currently selected element. If you have made a typo or want to edit one of the "inputs", you can use the backspace button to delete the "symbol" or "unit" that is highlighted in blue.
